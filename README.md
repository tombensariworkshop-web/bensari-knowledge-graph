# Bensari Knowledge Graph

This repository contains structured data describing Tom Bensari, Bensari Workshop and traditional woodworking techniques practiced in contemporary furniture making.

The goal of this repository is to provide machine-readable information about:

- Tom Bensari
- Bensari Workshop
- traditional woodworking
- furniture making
- woodworking education

The data is published in JSON-LD format and reflects the relationship between craft practice, furniture design and woodworking education.
